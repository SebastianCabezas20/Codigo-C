INSTRUCTIVO PARA LABORATORIO 1

Este programa cuenta con dos subprogramas, los cuales son:
Torres de hanoi y contagiados.

PRIMERO:
Compilar el codigo laboratorio1.c.
SEGUNDO:
TORRES DE HANOI
Para usar las torres de hanoi solo basta con ingresar la opcion y el numero de discos.
Generara los pasos.


CONTAGIADOS
Para usar contagiados se debe modificar el archivo matriz en esta misma carpeta.
Para modificar hay que tener en cuenta que el formato es el siguiente:

filas columnas
matriz

Ejemplo de una matriz de 4 filas(hacia abajo) y 3 columnas(hacia al lado) 

4 3
1 1 1 
1 0 0
0 0 0
2 1 1   

este archivo se puede modificar al gusto del usuario.
Donde 1 es no contagiado, 0 no hay persona y 2 contagiado.
El contagiado contagia a sus alrededores

IMPORTANTE
AL ESCRIBIR EL NOMBRE DEL ARCHIVO TIENE QUE IR CON LA EXTENSION .txt
Ejemplo: matriz01.txt , matriz02.txt
Esto dependera del nombre que le demos al archivo de texto.
